# frc
TEAM9214 First Robotics Competition Prototype space

## Projects included

`team9214_ws`
- Structure for the team autonomy codebase.  The includes localization based on Apriltag detection, interfacing with the RoboRIO, and executing behaviors with Behavior Trees.

`test_ros_roborio_comms`
- Application to setup python venv, deploy robot.py to the RoboRIO, and verifiying a ROS2 application can interface with robotpy through network tables.
